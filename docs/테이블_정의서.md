## 3. MEMBER

회원/관리자 계정.

| 컬럼 | 타입 | NULL | 기본값 | 설명 |
|------|------|------|--------|------|
| **MEMBER_ID** | VARCHAR(16) | NO | - | PK (MEMB-00001) |
| MEMBER_NM | VARCHAR(50) | YES | - | 회원명 |
| PASSWORD | VARCHAR(200) | YES | - | 비밀번호 (BCrypt 해시) |
| TEL | VARCHAR(500) | YES | - | 연락처 (AES-256-GCM 암호화) |
| TEL_LAST4 | VARCHAR(4) | YES | - | 연락처 뒷4자리 (평문, 검색용) |
| EMAIL | VARCHAR(100) | YES | - | 이메일 |
| USE_YN | CHAR(1) | YES | - | 사용여부 |
| REG_DT | DATETIME | YES | GETDATE() | 등록일시 |

**비고:**
- PASSWORD는 BCrypt 단방향 해시 (`$2a$` 접두어)
- TEL은 AES-256-GCM 양방향 암호화, TEL_LAST4는 검색용 평문

## 8. SCHEDULE

일정/회차 관리.

| 컬럼 | 타입 | NULL | 기본값 | 설명 |
|------|------|------|--------|------|
| **SCHEDULE_ID** | VARCHAR(16) | NO | - | PK (SCHD-00001) |
| PROGRAM_ID | VARCHAR(16) | YES | - | FK → PROGRAM |
| EVENT_DATE | DATE | YES | - | 행사 날짜 |
| EVENT_TIME | VARCHAR(10) | YES | - | 시작 시간 (HH:mm) |
| SESSION_NO | INT | YES | 1 | 회차 번호 (1, 2, 3...) |
| TOTAL_SEATS | INT | YES | - | 총 정원 (온라인+오프라인) |
| AVAIL_SEATS | INT | YES | - | 총 잔여석 (온라인+오프라인) |
| ONLINE_CAPACITY | INT | YES | 0 | 온라인 정원 (한도) |
| OFFLINE_CAPACITY | INT | YES | 0 | 오프라인 정원 (한도) |
| ONLINE_AVAIL | INT | YES | 0 | 온라인 잔여석 |
| OFFLINE_AVAIL | INT | YES | 0 | 오프라인 잔여석 |
| STATUS | VARCHAR(20) | YES | - | 상태 (OPEN / CLOSED / CANCELLED) |
| REG_DT | DATETIME | YES | GETDATE() | 등록일시 |

**정원 관계:**
- `TOTAL_SEATS = ONLINE_CAPACITY + OFFLINE_CAPACITY`
- `AVAIL_SEATS = ONLINE_AVAIL + OFFLINE_AVAIL`
- 온라인 예약 시 `ONLINE_AVAIL` 원자적 차감 (`WHERE ONLINE_AVAIL >= #{qty}`)

**현재 메뉴 구조:**

```

  ├── 프로그램 (MENU-00017)
  ├── 일정 (MENU-00003)
  └── 회차별 가격 (MENU-00016)
일반 입장권 (MENU-00018)
좌석 (MENU-00005)
예매 (MENU-00006)
회원 (MENU-00007)
시스템 관리 (MENU-00008) ── 그룹
  ├── 권한 (MENU-00009)
  ├── 메뉴 (MENU-00010)
  ├── 사용자권한 (MENU-00011)
  ├── 권한메뉴 (MENU-00012)
  └── 공통코드 (MENU-00013)
기초설정 (MENU-00014) ── 그룹
  ├── 권종관리 (MENU-00015)
  ├── 휴무일관리 (MENU-00019)
  ├── 영업시간 (MENU-00020)

```

| 컬럼 | 타입 | NULL | 기본값 | 설명 |

| **DISCOUNT_ID** | VARCHAR(16) | NO | - | PK (DCNT-00001) |

| START_DATE | DATE | YES | - | 적용 시작일 (NULL=제한없음) |
| END_DATE | DATE | YES | - | 적용 종료일 (NULL=제한없음) |
| APPLY_TIME_FROM | VARCHAR(5) | YES | - | 적용 시작시간 HH:mm (NULL=영업시작부터) |
| APPLY_TIME_TO | VARCHAR(5) | YES | - | 적용 종료시간 HH:mm (NULL=영업종료까지) |
| APPLY_DAYS | VARCHAR(20) | YES | - | 적용 요일 (예: 1,2,3,4,5=평일, NULL=전체) |
| APPLY_TICKET_TYPE_ID | VARCHAR(16) | YES | - | 적용 권종 FK → TICKET_TYPE (NULL=전체) |
| USE_YN | CHAR(1) | YES | Y | 사용여부 |
| REMARK | NVARCHAR(200) | YES | - | 비고 |
| REG_DT | DATETIME | YES | GETDATE() | 등록일시 |
